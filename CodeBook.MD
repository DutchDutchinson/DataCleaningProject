This codebook sumarizes the file: 'tidy.text'

The dataset is derived from the study Human Activity Recognition Using Smartphone Dataset.  The data was found at this URL: http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones


=========================================

There were two main identifiers: 

-subject: which is the ID of the participant
-activity: this was the activity the participant was participating in.  There were 6 possible activities: WALKING (value 1), WALKING_UPSTAIRS (value 2), WALKING_DOWNSTAIRS (value 3),
 SITTING (value 4), STANDING (value 5),LAYING (value 6)
